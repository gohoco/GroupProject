To execute the project, please follow the following steps.

1. Import the whole project into eclipse, notice that you may need to import the libraries to java project.

2. Run the Spider.java which contains the main method. After that the database called "searchEngine" will be generated.

3. Run the Test.java, after that, a spider_result.txt will be generated which contains all the information needed in Phase 1.

4. Run the Prescores.java, the idf of words, max tf and square root of document length of pages will be computed and stored.
